# Image Background Remover

## Building Source
- Make sure you have DirectX 11 SDK installed
- Open command prompt in project root
- Install vcpkg using 'git clone https://github.com/microsoft/vcpkg.git'
- Navigate to vcpkg folder using 'cd vcpkg'
- Run vcpkg bootstrap script using '.\bootstrap-vcpkg.bat'
- Move vcpkg.json to vcpkg/
- Install dependencies using 'vcpkg install'
- Open Project using Visual Studio, make sure the target platform is x64 and compile!
